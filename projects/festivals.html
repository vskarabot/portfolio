Festivals

all in one solution for music festival goers

Short Description

Since I was a little kid, I've always liked music - especially house music and it's subgenres. So naturally when I was old enough I started visiting music festivals. But one thing that always bothered me was how hard it is to find good festival related info. So I came up with an idea, that brings searching and exploring festivals into one place, combined with different options to connect with other festival goers and many other cool features that you'll see below.

Functionalities

The app supports following functionalities:

  • Discovering and searching festivals
  • Selected festival info
  • Nearby accomodation search
  • Route to festival planner
  • Dedicated forum for each festival
  • Public chat rooms
  • Finding people with same interests and contacting them with private chats

This functionalities provide user with complete solution in one place. All it's left is to buy the tickets at the official seller.

The next step was to create database that supports all the functionalities, with 18 tables in total, that we can split into 4 parts based on what their goal is:

  • User table - allows user to signup/login and also connect to other tables, so they can can like, post, comment...
  • Music festival tables - contain infos about festivals, user favourites, genres, user favourite genres, festival genres...
  • Forum tables - contain infos about posts, comments and who liked them.
  • Chat tables - contain chat infos, chat members, messages and notifications.
