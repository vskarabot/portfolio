# My Portfolio

This project showcases some of my personal info, previous work, skills and experience as a developer.

## 🌐 Live Site

Check it out here: [https://vskarabot.github.io/portfolio/]

## 🛠️ Built With

Portfolio website was built from scratch using only HTML, CSS, and vanilla JavaScript - without relying on external libraries or frameworks.

## ✨ Features

- Responsive grid design
- About me section
- Contact details
- Presentation of some of my biggest projects - Festivals, Birds, Hiking and Wordle
- Project: GTA VI Countdown, that shows the power of CSS for scroll animation - !!!Experimental -> Works in Edge, Chrome and (Firefox)
- Project: World Map, that uses Robinson Projection and displays countries I have visited in the past and allows users to play a game to list as much countries as possible
- Live digital clock, with CSS created roman clock in the background
- Theme selection
- Link to my SoundCloud, with CSS frequency audio animation
